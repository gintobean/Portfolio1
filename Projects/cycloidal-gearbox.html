9:1 Cycloidal Drive

Introduction

This project involved the integration and control of mechanical and electrical systems. O ur team strived to build a thrust vector control system that was robust and reusable, and could be mass-manufactured. Many physical and software tests had to be run to optimize rocket stability, considering rockets launch at speeds of over 300m/s. Thus the design allows for sensitive real-time calculations and a sizable amount of torque to change the rocket’s trajectory in mid-flight.

Initial Concept

Inspired by the spaceX Falcon 9 launch, I set off to design and prototype my own thrust vectoring system for model rockets. The first design allowed the engine mount to rotate via two universal joints and two longerons attached on the body tube. However, this design didn’t seem stable enough, so we decided to use a ball joint to hold the mount in place and ensure larger and smoother range of motion. Servo motors rotated each longeron.

Prototype

I modeled the prototype via solidworks and created several animations to spatially understand the movement. Kinematic analysis was done via matlab to find how rotating the servo motors would change the roll and pitch of the engine mount. This allowed us to draft the skeleton code for the P&ID control system, which would allow the flight computer to adjust the engine mount tilt in real time.


After calculating optimal dimensions for the 6-bar mechanism, I utilized solidworks to model the mechanism and print parts. The full print was in PLA. However, the grooves on the PLA presented the printed gears from smoothly meshing, so I purchased acrylic gears of the same dimensions. The result is pictured below: The final prototype consisted of acrylic gears and PLA linkages. An arm was added to the pinion to allow the attachment of a universal joint and steering wheel. This steering mechanism will be part of a larger electric car project.

Conclusion

In reality, most steering mechanisms will not function on acrylic and PLA due to high friction and thus heavy wear. However, this project provides insight on various, more complicated mechanisms that can be used in place of simpler, less accurate mechanisms, possibly eliminating the need for variable steering and thus reducing manufacturing costs.
